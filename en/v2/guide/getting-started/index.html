Getting Started

Welcome to our documentation! This guide will help you get started with our platform.

This is the documentation for version 2.0. If you're using version 1.0, please click here.

Prerequisites

  • Node.js 18 or higher
  • npm, yarn, or pnpm
  • Basic knowledge of JavaScript

Installation

You can install our package using npm:

npm install @our-package/core@2.0.0

Or using yarn:

yarn add @our-package/core@2.0.0

Or using pnpm:

pnpm add @our-package/core@2.0.0

Basic Usage

Here's a simple example of how to use our package:

import { createApp } from '@our-package/core';

// New in v2: Async/await support
async function initializeApp() {
  const app = await createApp({
    // Configuration options
    debug: true,
    mode: 'development',
    // New in v2: Enhanced options
    plugins: ['logger', 'analytics'],
    performance: {
      monitoring: true,
      metrics: ['cpu', 'memory']
    }
  });

  await app.start();
}

initializeApp();

New Features in v2

  • Full async/await support
  • Enhanced plugin system
  • Performance monitoring
  • Improved error handling
  • TypeScript type definitions included
