Getting Started

Welcome to our documentation! This guide will help you get started with our platform.

Prerequisites

  • Node.js 18 or higher
  • npm, yarn, or pnpm
  • Basic knowledge of JavaScript

Installation

You can install our package using npm:

npm install @our-package/core

Or using yarn:

yarn add @our-package/core

Or using pnpm:

pnpm add @our-package/core

Basic Usage

Here’s a simple example of how to use our package:

import { createApp } from '@our-package/core';

const app = createApp({
  // Configuration options
  debug: true,
  mode: 'development'
});

app.start();
