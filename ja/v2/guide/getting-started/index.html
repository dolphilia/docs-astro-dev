はじめに

ドキュメントへようこそ！このガイドでは、プラットフォームの使い方について説明します。

これはバージョン2.0のドキュメントです。バージョン1.0をお使いの場合は、こちらをクリックしてください。

前提条件

  • Node.js 18以上
  • npm、yarn、またはpnpm
  • JavaScriptの基本的な知識

インストール

npmを使用してパッケージをインストールできます：

npm install @our-package/core@2.0.0

または、yarnを使用：

yarn add @our-package/core@2.0.0

または、pnpmを使用：

pnpm add @our-package/core@2.0.0

基本的な使い方

パッケージの使用例：

import { createApp } from '@our-package/core';

// v2の新機能: Async/awaitサポート
async function initializeApp() {
  const app = await createApp({
    // 設定オプション
    debug: true,
    mode: 'development',
    // v2の新機能: 拡張オプション
    plugins: ['logger', 'analytics'],
    performance: {
      monitoring: true,
      metrics: ['cpu', 'memory']
    }
  });

  await app.start();
}

initializeApp();

v2の新機能

  • 完全なasync/awaitサポート
  • 拡張プラグインシステム
  • パフォーマンスモニタリング
  • 改善されたエラーハンドリング
  • TypeScript型定義の同梱

  • APIリファレンスを調べる
