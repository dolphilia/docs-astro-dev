はじめに

ドキュメントへようこそ！このガイドでは、プラットフォームの使い方について説明します。

前提条件

  • 設定オプションについて詳しく学ぶ
  • APIリファレンスを探索する
  • いくつかのサンプルを確認する

インストール

npmを使用してパッケージをインストールできます：

npm install @our-package/core

または、yarnを使用：

yarn add @our-package/core

または、pnpmを使用：

pnpm add @our-package/core

基本的な使い方

パッケージの使用例：

import { createApp } from '@our-package/core';

const app = createApp({
  // 設定オプション
  debug: true,
  mode: 'development'
});

app.start();

次のステップ

基本的なセットアップが完了したら、次のことができます：

  • 設定オプションについてさらに学ぶ
  • APIリファレンスを調べる
  • いくつかのサンプルを確認する
