# タブコンポーネントの例

このページでは、ドキュメントでのタブコンポーネントの使用方法を示します。

## 基本的な使い方

<Tabs>
  <TabItem label="HTML">
    ```html
    <!DOCTYPE html>
    <html>
      <head>
        <title>Hello World</title>
      </head>
      <body>
        <h1>Hello, world!</h1>
      </body>
    </html>
    ```
  </TabItem>
  <TabItem label="CSS">
    ```css
    body {
      font-family: system-ui, sans-serif;
      color: #333;
      background-color: #f9f9f9;
    }
    
    h1 {
      color: navy;
    }
    ```
  </TabItem>
  <TabItem label="JavaScript">
    ```js
    document.addEventListener('DOMContentLoaded', () => {
      console.log('Hello, world!');
    });
    ```
  </TabItem>
</Tabs>

## 同期タブ

これらのタブは `syncKey` プロパティを使用してページ全体で同期されます。

<Tabs syncKey="languages">
  <TabItem label="JavaScript">
    ```js
    // JavaScript の例
    function greet(name) {
      return `こんにちは、${name}さん！`;
    }
    
    console.log(greet('世界'));
    ```
  </TabItem>
  <TabItem label="TypeScript">
    ```ts
    // TypeScript の例
    function greet(name: string): string {
      return `こんにちは、${name}さん！`;
    }
    
    console.log(greet('世界'));
    ```
  </TabItem>
  <TabItem label="Python">
    ```python
    # Python の例
    def greet(name):
        return f"こんにちは、{name}さん！"
    
    print(greet("世界"))
    ```
  </TabItem>
</Tabs>

同じ `syncKey` を持つ別のタブセットは、上記のタブと同期状態を維持します：

<Tabs syncKey="languages">
  <TabItem label="JavaScript">
    JavaScriptは、Webページで動的なコンテンツを可能にするスクリプト言語です。
  </TabItem>
  <TabItem label="TypeScript">
    TypeScriptは、JavaScriptをベースにした強力な型付けプログラミング言語です。
  </TabItem>
  <TabItem label="Python">
    Pythonは、高水準の汎用プログラミング言語です。
  </TabItem>
</Tabs>
