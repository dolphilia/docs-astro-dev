# アイコンコンポーネントの例

アイコンコンポーネントを使用すると、ドキュメント内にさまざまなアイコンを表示できます。

## 基本的な使い方

アイコンコンポーネントの基本的な使用例：

<Icon name="info" />

## アイコンのサイズ

アイコンにはさまざまなサイズを指定できます：

<div style="display: flex; align-items: center; gap: 1rem;">
  <Icon name="info" size={16} />
  <Icon name="info" size={24} />
  <Icon name="info" size={32} />
  <Icon name="info" size={48} />
</div>

## アイコンの色

アイコンの色をカスタマイズできます：

<div style="display: flex; align-items: center; gap: 1rem;">
  <Icon name="info" color="#007bff" />
  <Icon name="warning" color="#ffc107" />
  <Icon name="error" color="#dc3545" />
  <Icon name="check" color="#28a745" />
</div>

## テキスト内でのアイコンの使用

アイコンはテキスト内でインラインで使用できます：

これは情報 <Icon name="info" /> メッセージです。

これは警告 <Icon name="warning" color="#ffc107" /> メッセージです。

これはエラー <Icon name="error" color="#dc3545" /> メッセージです。

これは成功 <Icon name="check" color="#28a745" /> メッセージです。
