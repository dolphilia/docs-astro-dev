# Extended Icon Components

This guide demonstrates how to use the extended icon components in your documentation.

## Built-in Icons

The basic `Icon` component provides a set of built-in icons:

<div style="display: flex; align-items: center; gap: 1rem;">
  <Icon name="info" />
  <Icon name="warning" />
  <Icon name="error" />
  <Icon name="success" />
</div>

For more details on the built-in icons, see the [Icons Example](/en/v1/guide/icons-example) page.

## Lucide Icons

The `LucideIcon` component allows you to use any icon from the [Lucide icon library](https://lucide.dev/icons/).

### Basic Usage

<LucideIcon name="activity" />

### Icon Sizes

You can specify different sizes for icons:

<div style="display: flex; align-items: center; gap: 1rem;">
  <LucideIcon name="activity" size="1em" />
  <LucideIcon name="activity" size="1.5em" />
  <LucideIcon name="activity" size="2em" />
  <LucideIcon name="activity" size="3em" />
</div>

### Icon Colors

You can customize the color of icons:

<div style="display: flex; align-items: center; gap: 1rem;">
  <LucideIcon name="heart" color="#ff0000" />
  <LucideIcon name="star" color="#ffd700" />
  <LucideIcon name="thumbs-up" color="#008000" />
  <LucideIcon name="bell" color="#0000ff" />
</div>

### Stroke Width

You can adjust the stroke width of Lucide icons:

<div style="display: flex; align-items: center; gap: 1rem;">
  <LucideIcon name="activity" strokeWidth={1} />
  <LucideIcon name="activity" strokeWidth={2} />
  <LucideIcon name="activity" strokeWidth={3} />
</div>

## FontAwesome Icons

The `FontAwesomeIcon` component allows you to use any icon from the [FontAwesome icon library](https://fontawesome.com/icons).

### Basic Usage

<FontAwesomeIcon name="coffee" />

### Icon Styles

FontAwesome provides different icon styles through prefixes:

<div style="display: flex; align-items: center; gap: 1rem;">
  <FontAwesomeIcon name="star" prefix="fas" /> {/* Solid */}
  <FontAwesomeIcon name="star" prefix="far" /> {/* Regular */}
  <FontAwesomeIcon name="github" prefix="fab" /> {/* Brands */}
</div>

### Icon Sizes

You can specify different sizes for icons:

<div style="display: flex; align-items: center; gap: 1rem;">
  <FontAwesomeIcon name="coffee" size="1em" />
  <FontAwesomeIcon name="coffee" size="1.5em" />
  <FontAwesomeIcon name="coffee" size="2em" />
  <FontAwesomeIcon name="coffee" size="3em" />
</div>

### Icon Colors

You can customize the color of icons:

<div style="display: flex; align-items: center; gap: 1rem;">
  <FontAwesomeIcon name="heart" color="#ff0000" />
  <FontAwesomeIcon name="star" color="#ffd700" />
  <FontAwesomeIcon name="thumbs-up" color="#008000" />
  <FontAwesomeIcon name="bell" color="#0000ff" />
</div>

## Using Icons in Text

Icons can be used inline with text:

This is an information <Icon name="info" /> message.

This is a warning <LucideIcon name="alert-triangle" color="#ffc107" /> message.

This is an error <FontAwesomeIcon name="exclamation-circle" color="#dc3545" /> message.

## Accessibility

For accessibility purposes, you can provide a label for the icon:

<LucideIcon name="info" label="Information" />

<FontAwesomeIcon name="info-circle" label="Information" />

When a label is provided, the icon will be announced by screen readers. Without a label, the icon is treated as decorative and will be ignored by screen readers.
