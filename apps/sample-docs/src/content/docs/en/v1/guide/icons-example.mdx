# Icon Component Examples

The Icon component allows you to display various icons in your documentation.

## Basic Usage

Here's a basic example of using the Icon component:

<Icon name="info" />

## Icon Sizes

You can specify different sizes for icons:

<div style="display: flex; align-items: center; gap: 1rem;">
  <Icon name="info" size={16} />
  <Icon name="info" size={24} />
  <Icon name="info" size={32} />
  <Icon name="info" size={48} />
</div>

## Icon Colors

You can customize the color of icons:

<div style="display: flex; align-items: center; gap: 1rem;">
  <Icon name="info" color="#007bff" />
  <Icon name="warning" color="#ffc107" />
  <Icon name="error" color="#dc3545" />
  <Icon name="check" color="#28a745" />
</div>

## Using Icons in Text

Icons can be used inline with text:

This is an information <Icon name="info" /> message.

This is a warning <Icon name="warning" color="#ffc107" /> message.

This is an error <Icon name="error" color="#dc3545" /> message.

This is a success <Icon name="check" color="#28a745" /> message.

## Accessibility

For accessibility purposes, you can provide a label for the icon:

<Icon name="info" label="Information" />
