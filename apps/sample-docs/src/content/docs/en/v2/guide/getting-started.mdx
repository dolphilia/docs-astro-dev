# Getting Started with v2

Welcome to version 2 of our documentation! This guide will help you get started with our platform.

## Prerequisites

- Node.js 18 or higher
- npm, yarn, or pnpm
- Basic knowledge of JavaScript and TypeScript

## Installation

You can install our package using npm:

```bash
npm install @our-package/core@2
```

Or using yarn:

```bash
yarn add @our-package/core@2
```

Or using pnpm:

```bash
pnpm add @our-package/core@2
```

## Basic Usage

Here's a simple example of how to use our package with TypeScript:

```typescript
import { createApp, type AppOptions } from '@our-package/core';

const options: AppOptions = {
  // Configuration options
  debug: true,
  mode: 'development',
  // New in v2
  plugins: [],
  theme: 'light'
};

const app = createApp(options);

app.start();
```

## New Features in v2

Version 2 introduces several new features:

- **TypeScript Support**: Full TypeScript support with type definitions
- **Plugin System**: Extend functionality with plugins
- **Theming**: Built-in theme support
- **Performance Improvements**: 50% faster than v1
